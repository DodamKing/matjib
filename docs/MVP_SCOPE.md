# MVP 범위

목표: **실데이터 연동까지 동작하는 풀스택**. 아래 경계를 넘는 기능은 v2로 미룬다.

## 만드는 것 (IN)

- [ ] 원클릭 진입: 위치 동의 → 즉시 결과 (가입/로그인 없음)
- [ ] 도보 시간 필터: 5분(300m) / 10분(600m) / 15분(1km)
- [ ] 소상공인 API 연동: 반경 내 음식점 조회 (실데이터)
- [ ] 카드 3개 UI (Zero-Scroll, 모바일 풀스크린)
- [ ] 셔플: 같은 후보 풀에서 새 3개 재추첨
- [ ] 자연어 상황 입력 → Claude로 업종 매칭
- [ ] "AI 처방" 페르소나 카피
- [ ] 카드에서 길찾기 링크 (카카오 보강 — 되면 표시)

## 안 만드는 것 (OUT — 의식적으로 제외)

스코프 확장 = 토큰·시간 낭비. 아래는 v1에서 손대지 않는다.

- ❌ 회원가입 / 로그인 / 마이페이지
- ❌ 리뷰 / 평점 / 별점 (데이터에 없음 + USP와 충돌 — DECISIONS D3)
- ❌ 즐겨찾기 / 아카이빙 / 방문 기록
- ❌ 예약 / 주문 / 결제 연동
- ❌ 지도 화면 (핀 찍힌 인터랙티브 맵)
- ❌ 리스트 / 무한스크롤 / 더보기
- ❌ 광고 / 스폰서 / 상위노출 슬롯
- ❌ 평점 스크래핑 등 외부 품질 데이터 수집

## 완료 기준 (Definition of Done)

1. 모바일 브라우저에서 링크 접속 → 위치 동의 → 3개 카드가 뜬다.
2. 셔플 누르면 다른 3개가 뜬다.
3. 자연어 입력 시 업종이 바뀐 결과가 나온다.
4. 카드의 거리/도보시간이 실제 좌표 기반으로 맞다.
5. API 키가 클라이언트 번들에 노출되지 않는다.
